h1. Queriesh2. /queries.:formath3. GETReturns the list of all custom queries visible by the user (public and private queries) for all projects.+Examples+:<pre>GET /queries.xml</pre>+Response+:<pre><?xml version="1.0" encoding="UTF-8"?><queries type="array" total_count="5" limit="25" offset="0">  <query>    <id>84</id>    <name>Documentation issues</name>    <is_public>true</is_public>    <project_id>1</project_id>  </query>  <query>    <id>1</id>    <name>Open defects</name>    <is_public>true</is_public>    <project_id>1</project_id>  </query></queries></pre>Armed with a query id, you can get the corresponding issue list using:<pre>GET /issues.xml?query_id=:idGET /issues.xml?query_id=:id&project_id=foo</pre>
